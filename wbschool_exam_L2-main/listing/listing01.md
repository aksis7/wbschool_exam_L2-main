Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```
[77 78 79]


Ответ:
```
Слайс a[1:4] включает элементы массива с индексами 1, 2 и 3 (77, 78, 79) и исключает элемент с индексом 4.

```
